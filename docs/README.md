# Status Bar Playback Controls
Status Bar Playback Controls Blender add-on

### About
**Status Bar Playback Controls** is a simple Blender add-on that allows user to control animation playback at all times without having **Timeline** open.

### Installation
1. Download a version from https://github.com/Pazelock/Status-Bar-Playback-Controls/tree/main/Status-Bar-Playback-Controls
2. Drag n' drop downloaded .zip file into Blender.
3. Enable the add-on by clicking **OK** in a pop-up.

### Usage
In the middle of the status bar you can see the input field with current frame number on it. This replicates Blender's built-in interactive frame indicator.
Press **space** to start animation playback. You'll see a few buttons appear on your status bar. They act exactly as built-in do. Three buttons you see by default are not all of them. If you want to enable the rest, please do so in add-on preferences.

### Preferences and Personalization
To find the add-on preferences, go to:

`Edit -> Preferences -> Add-ons`

Type **"Status Bar Playback Controls"** in the search field. Click the arrow to expand the add-on information and access **preferences**.
In the **Preferences** section, you will find checkboxes to enable / disable buttons:
- Jump to First Frame
- Previous Keyframe *(disabled by default)*
- Play in Reverse / Pause *(disabled by default)*
- Play / Pause
- Next Keyframe *(disabled by default)*
- Jump to Last Frame
- Current Frame Input
- Animation Player Cancel Button *(disabled by default)*
